About this System

This is a minimal static sports management system built using only HTML and CSS. It demonstrates simple navigation, event listing, and contact form layout without backend logic.

How to use

  1. Open events.html to view the sample event list.
  2. Use contact.html to view the contact form (static).
  3. Edit the HTML files to add more teams, events, and pages.
