Hotel Management - All Assignments (simple)
Folders:
- Assignment1_Static: static pages (index, rooms, about, contact)
- Assignment2_Responsive: responsive rooms page (media queries)
- Assignment3_JSValidation: booking validation demo
- Assignment4_ArraySearch: JS array + search demo
- Assignment5_PHP_MySQL: booking form (PHP) + save script + config + create_tables.sql
- Assignment6_Admin: admin login & dashboard (UI only)
- Assignment7_BookingList: bookings list (PHP) reading DB from Assignment5

How to use PHP parts (Assignments 5 & 7):
1. Install PHP & MySQL locally (XAMPP, WAMP, LAMP).
2. Place the entire folder in your server's web root (e.g., htdocs for XAMPP).
3. Run the SQL in create_tables.sql to create the database and 'bookings' table.
4. Edit Assignment5_PHP_MySQL/config.php to set DB username/password if needed.
5. Open booking_form.php to submit bookings. View bookings.php to see saved bookings.
